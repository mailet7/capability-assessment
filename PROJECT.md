# Capability Path Website

## Purpose

Capability Path helps parents develop capable, self-directed, adaptable human beings.

The website exists to help parents understand the framework, assess development, identify next steps, and implement the system in daily life.

---

The assessment is the primary entry point into Capability Path.

# Layer System

## Layer 1

Minimum Day

The system succeeds if these happen:

### Body

Morning routine

### Will

One completion

### Direction

Home reset

Use when:

- Overwhelmed
- Sick
- Low energy
- Resistance is high
- Life is chaotic

---

## Layer 2

Add:

### Mind

Learning block

### Direction

Independent Time

Purpose:

Develop capability without overwhelming the family.

---

## Layer 3

Expansion

Examples:

- Projects
- Leadership
- Initiative
- Independent learning
- Contribution

Complexity is added only after foundations become stable.

# Development Philosophy

Development follows a progression.

Body supports Will.

Will supports Mind.

Mind supports Direction.

The framework is not rigid.

All areas matter.

However, foundations tend to support everything above them.

---

# Assessment Philosophy

Assessment exists to identify:

"What needs to develop next?"

Not:

"What's wrong?"

The assessment should reduce overwhelm and create clarity.

---

# Road Map Philosophy

The goal is not perfection.

The goal is progress.

Every developmental area progresses through increasing capability.

Movement occurs through:

- Appropriate expectations
- Environment design
- Repeated practice
- Gradual transfer of responsibility

---

# Leadership Philosophy

Capability Path encourages leadership rather than management.

Managers direct behavior.

Leaders develop capability.

The goal is not permanent supervision.

The goal is increasing independence.

---

# Independent Time Philosophy

Independent Time is not free time.

Independent Time develops:

- Initiative
- Creativity
- Responsibility
- Self-direction

Children learn to direct themselves.
